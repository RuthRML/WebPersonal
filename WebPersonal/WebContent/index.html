A continuación expongo algunos de los proyectos llevados a cabo durante el Grado de Ingeniería Informática. Mi perfil en GitHub contiene la mayoría de ellos.

Aplicación Shazam sin interfaz.

Se trata de un programa capaz de reconocer las frecuencias de las canciones y clasificarlas para luego poder compararlas. Este proyecto está desarrollado en Java y no cuenta con interfaz gráfica. Es necesario que se registren las canciones antes de que el programa pueda identificarlas. El principal objetivo es comprender cómo se manejan las frecuencias de datos multimedia como la música.

Juego web de La Oca.

Una aplicación web que implementa el tradicional juego de La Oca. Está desarrollada en Java, junto con HTML, JavaScript, JQuery y HTML y usa MongoDB, una base de datos NoSQL.
